# `tmux` Key Bindings

Action Key: `Ctrl-b`

## Sessions

- Rename session: `$`
- Detach session: `D`
- Next session: `)`
- Previous session: `(`

## Windows

- Create new window: `c`
- Move to next window: `n`
- Move to previous window: `p`
- Move to window last used: `l`
- Select window by number: `0-9`
- Select window by name: `'`
- Change window number: `.`
- Rename window: `r`
- Search windows: `f`
- Kill window: `&`

## Panes

- Vertical split: `%`
- Horizontal split: `"`
- Move pane right: `Right Arrow`
- Move pane left: `Left Arrow`
- Move pane up: `Up Arrow`
- Move pane down: `Down Arrow`
- Go to next pane: `o`
- Go to last active pane: `;`
- Move pane right: `}`
- Move pane left: `{`
- Convert pane to window: `!`
- Kill pane: `x`

## Copy Mode

### Key Bindings

- Enter copy mode: `[`
- Paste from buffer: `]`

### Commands

- Start selection: `Space`
- Copy selection: `Enter`
- Clear selection: `Escape`
- Go to top: `g`
- Go to bottom: `G`
- Move cursor left: `h`
- Move cursor right: `l`
- Move cursor up: `k`
- Move cursor down: `j`
- Search: `/`
- List paste buffers: `#`
- Quit: `q`
